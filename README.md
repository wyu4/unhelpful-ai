# The UnHelpful AI
The Unhelpful AI Assistant, at your service!

## How to run
1) Install Node.js / npm [here](https://nodejs.org/en/download).
2) Install libraries
```bash
npm install
```
2) Install Netlify
```bash
npm install -g netlify-cli
```
4) Run the server
```bash
netlify dev
```

## Secrets
Please set the environment variable `SCRAPYARD_KEY` to your OpenAI API key.
