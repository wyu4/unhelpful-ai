# The UnHelpful AI
The Unhelpful AI Assistant, at your service! <br>
This was made for the [Scrapyard](https://scrapyard.hackclub.com/) Hackathon.

## How to run
1) Install Node.js / npm [here](https://nodejs.org/en/download).
2) Install libraries
```bash
npm install
```
2) Run the server
```bash
npm start
```
